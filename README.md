# SpringBoot-with-S3

API con SpringBoot para interactuar con AWS S3

- Crear buckets en S3.
- Verificar si un bucket existe.
- Listar todos los buckets.
- Subir archivos a un bucket.
- Descargar archivos desde un bucket y guardarlos en una carpeta local.
- Generación de URLs pre-firmadas para subir archivos.
- Generación de URLs pre-firmadas para descargar archivos.
